# 🎧 DUPA CRM v3

CRM de booking DJ pour Düparecords — connecté à Firebase + n8n.

## Stack
- **Frontend** : HTML/CSS/JS vanilla (single-file)
- **Base de données** : Firebase Firestore
- **Automatisation** : n8n Cloud (scraping + outreach)
- **IA** : Claude API (Anthropic)
- **Hébergement** : Netlify

## Déploiement
1. Clone ce repo
2. Configure Firebase dans `index.html` (section FIREBASE_CONFIG)
3. Déploie sur Netlify (auto-deploy depuis GitHub)

## Fonctionnalités
- 🏢 Gestion des prospects (clubs, festivals, bars)
- 🔍 Scraping automatique via n8n + Google Places
- 📨 Outreach multi-canal (Email, Instagram, LinkedIn, WhatsApp)
- 🤖 IA intégrée (analyse, personnalisation messages)
- 📊 Pipeline de vente visuel
- 🎧 Gestion du roster DJs
